# 📱 Android Study
> 모바일 프로그래밍 수업에서 진행한 안드로이드 기초 실습 코드 및 학습 내용을 정리한 저장소입니다.<br/>
> 이곳에서 학습한 내용은 이후 메인 프로젝트인 <b>[Deokmoa (덕모아)](https://github.com/seohyun27/android-deokmoa)</b>의 개발로 이어졌습니다.

## 📂 Contents

### 📁 project_docs
- **프로젝트 초기 기획안**
  - 메인 프로젝트인 '덕모아' 개발을 위한 아이디어 스케치 및 초기 기획 문서들을 모아두었습니다.

### 📁 Ch6_View
- **로그인 화면 (Login Screen)**
  - 버튼(Button)과 다양한 레이아웃을 활용하여 간단한 로그인 화면을 구현했습니다.
  - 드래그 앤 드롭 방식이 아닌 **XML 코드를 직접 작성**하여 UI를 구성하는 연습을 진행했습니다.

### 📁 Ch7_Layout
- **전화 키패드 (Phone Dial Pad)**
  - `TextView`와 `ImageView`를 조합하여 전화 걸기 화면을 구현했습니다.
  - 안드로이드 스튜디오의 **Visual Editor(Drag-and-drop)** 기능을 활용해 UI를 배치하는 방법을 익혔습니다.

### 📁 Ch8_Event
- **스톱워치 (Stopwatch)**
  - 간단한 스톱워치 애플리케이션입니다.
  - <b>이벤트 핸들링(Event Handling)</b>을 통해 XML 레이아웃의 UI 컴포넌트와 Kotlin 로직(.kts)을 연결하고 제어하는 과정을 학습했습니다.
