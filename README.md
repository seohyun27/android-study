# 📱 Android Study
모바일 프로그래밍 수업에서 진행한 안드로이드 기초 실습 코드 및 학습 내용을 정리한 저장소입니다.  

## 🛠 Tech Stack
![Kotlin](https://img.shields.io/badge/kotlin-%237F52FF.svg?style=flat&logo=kotlin&logoColor=white) ![Android Studio](https://img.shields.io/badge/Android%20Studio-3DDC84.svg?style=flat&logo=android-studio&logoColor=white)

## 📂 Contents

### 📁 Ch6_View
- **로그인 화면 (Login Screen)**
  - 버튼(Button)과 다양한 레이아웃을 활용하여 간단한 로그인 화면을 구현했습니다.
  - 드래그 앤 드롭 방식이 아닌 **XML 코드를 직접 작성**하여 UI를 구성하는 연습을 진행했습니다.

### 📁 Ch7_Layout
- **전화 키패드 (Phone Dial Pad)**
  - `TextView`와 `ImageView`를 조합하여 전화 걸기 화면을 구현했습니다.
  - 안드로이드 스튜디오의 **Visual Editor(Drag-and-drop)** 기능을 활용해 UI를 배치하는 방법을 익혔습니다.

### 📁 Ch8_Event
- **스톱워치 (Stopwatch)**
  - 간단한 스톱워치 애플리케이션입니다.
  - **이벤트 핸들링(Event Handling)**을 통해 XML 레이아웃의 UI 컴포넌트와 Kotlin 로직(.kts)을 연결하고 제어하는 과정을 학습했습니다.
